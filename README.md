# Project Olive — weboldal

Prémium, kétnyelvű (HU/EN) bemutató oldal a budapesti Project Olive lakáshoz.
Egyetlen önálló `index.html` — nincs build, nincs függőség.

## Fotók hozzáadása

Másold a képeket az `images/` mappába az alábbi neveken. Amíg egy fájl hiányzik,
elegáns placeholder látszik a helyén — a kép bemásolása után azonnal megjelenik.

| Fájlnév | Mi kerüljön bele |
|---|---|
| `hero.jpg` | nyitókép (legszebb nappali-fotó, fekvő) |
| `living-1.jpg` | nappali, nagy fekvő kép |
| `living-2.jpg` | nappali, panorámás sarok |
| `living-3.jpg` | nappali, álló formátumú részlet |
| `kitchen-1.jpg` | konyha, álló |
| `kitchen-2.jpg` | konyhai részlet, fekvő |
| `gallery-1.jpg` | galéria-hálószoba éjszaka, álló |
| `gallery-2.jpg` | szabadon álló kád, fekvő |
| `shaft-1.jpg` | fényakna, álló |
| `bedroom-1.jpg` | második hálószoba, fekvő |
| `location-1.jpg` | utcakép / környék, álló |
| `story-1.jpg` | werkfotó a felújításból, álló |
| `creators-1.jpg` | az alkotók portréja, álló |
| `before-1.jpg` | előtte-fotó (ugyanaz a beállás, mint az utána!) |
| `after-1.jpg` | utána-fotó |
| `video-poster.jpg` | videó előnézeti kép |
| `olive-video.mp4` | a lakásvideó (MP4/H.264) |

Ajánlott méret: 1600–2400 px szélesség, tömörített JPG (~200–500 KB).
